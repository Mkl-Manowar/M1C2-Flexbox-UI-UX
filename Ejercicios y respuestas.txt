¿Cuál es un objetivo primordial?
Es el objetivo principal por el que se hace el proyecto, por ejemplo, el objetivo primordial de una app de mensajeria, seria poder enviar y recibir mensajes de texto entre dos personas.

¿Qué son los wireframes de baja fidelidad?
Son bocetos que se usan para saber el aspecto basico que quieres que tenga la web o aplicación, muestran de forma basica el aspecto y funciones de estas.

¿Qué es Git?
Es una herramienta que se utiliza para importar y exportar codigo de un projecto, guardando copias de versiones anteriores del proyecto para recuperar en caso de error.
Git permite a varias personas contribuir a un proyecto al mismo tiempo, añadiendo cada uno su propio codigo.


¿Qué es un sitemap?
Un diagrama que hace de mapa de la app, mostrando las diferentes rutas que hay para y donde llegan dependiendo que botones o enlaces pulses.
